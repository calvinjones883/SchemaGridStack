# SchemaGridStack - Multi-Layer Data Generator

SchemaGridStack uses stacked schema definitions to simulate complex datasets for testing, dev, or analytics.

## Features
- Layered schema system  
- CSV / JSON / SQL export  
- Fast CLI usage
